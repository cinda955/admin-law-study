⚔️ 警械武器条例学习

17条法条，年年考8-9题，性价比超高！

📜 模块1: 条例概述

核心概念：《人民警察使用警械和武器条例》是国务院制定的行政法规，一共才17条！别看它短，每年考试要考8-9道题！性价比超高！

📌 一、立法目的（第1条）

为什么要制定这个条例？想象一下：警察拿着枪、警棍，这些东西威力很大！如果乱用，分分钟伤害无辜群众！所以必须把警察使用警械武器的权力关进笼子里！

条例的目的是：

• 依法履行职责：警察执法要有法可依

• 正确使用警械武器：不是让你随便用，而是规范使用

• 维护公共安全和秩序：保护社会

• 保障公民合法权益：不能伤害无辜！

🌰 举例：警察抓小偷，小偷反抗。警察可以依法使用警械制服小偷，但不能把小偷偷打一顿！因为要保障公民合法权益！

📌 二、条例基本信息

• 制定机关：国务院

• 条文数量：17条

• 性质：行政法规

• 施行时间：1995年

🌰 举例：考试常考：这部条例是谁制定的？答案就是国务院！别和公安机关搞混了！

📌 三、警察使用警械武器受保护（第5条）

警察依法使用警械武器，是受法律保护的！什么意思？就是你配合执法，不要妨碍，否则警察可以依法采取措施！

🌰 举例：有人阻碍警察执法，推搡警察。这时候警察可以依法使用警械。因为警察依法执法受保护！

记忆口诀：条例虽短17条，考试8-9题跑不掉！立法目的是保护，既保执法又保公民！国务院制定要记牢！

⚠️ 易错点：有人以为是公安部制定的，错！是国务院制定的！

🔫 模块2: 警械和武器的分类

核心概念：这是年年必考的知识点！必须分清什么是警械、什么是武器！

📌 一、什么是警械（第3条）

警械就是警察按规定装备的非致命性器械！简单说就是：一般不致命！

具体包括：

• 驱逐性警械：警棍、催泪弹、高压水枪、特种防暴枪

• 约束性警械：手铐、脚镣、警绳

• 震慑性警械：警笛、警报器、红色回旋警灯

• 自卫性警械：防弹衣、头盔、盾牌

🌰 举例：警察抓人时给对方戴上手铐→约束性警械！追逃跑的人时用警棍→驱逐性警械！

📌 二、什么是武器（第3条）

武器就是警察按规定装备的致命性警用器械！说白了就是：枪支弹药！

记住这个：武器 = 枪支弹药 = 致命性！

🌰 举例：警察腰间的手枪、子弹，就是武器！防爆枪虽然有个枪字，但属于警械的驱逐性警械！

📌 三、警械和武器的核心区别

对比项警械武器
本质非致命性致命性
代表物品警棍、手铐、催泪弹枪支、弹药
使用频率较高（日常执法）较低（严重情况）
使用条件相对宽松严格限制

记忆口诀：警械武器分清楚：武器就是枪和弹，警棍手铐催泪弹！武器致命警械非，分类考试年年追！口诀：武器=枪弹，警械=棍铐绳弹！

⚠️ 易错点：特种防暴枪是警械不是武器！别看到枪字就以为是武器！

⚖️ 模块3: 使用原则（第4条）

核心原则：这个原则贯穿整个条例！后面所有条款都从这里派生出来！必须牢牢记住！

📌 一、两大原则

原则一：以制止违法犯罪行为为目的

使用警械武器的唯一目的是：制止违法犯罪！不是为了报复、不是为了打人！

🌰 举例：小偷已经停止盗窃，乖乖投降了。这时候警察就不能再动手打他了！因为目的已经达到！

原则二：尽量减少人员伤亡和财产损失

能用轻的解决，就不要用重的！层层递进！

📌 二、使用强度的递进原则（从轻到重）

记住这个顺序：

1. 口头制止：喊话、警告（最轻）

2. 徒手制止：不用工具，徒手控制

3. 使用警械：警棍、催泪弹等

4. 使用武器：枪支（最重！）

🌰 举例：有人闹事，警察先喊话：别闹了！对方不听，警察上去徒手按住他。还是不服，警察拿出警棍。还是不行，最后才开枪！这就叫从轻到重！

📌 三、必须先警告（第7条、第9条）

使用警械武器之前，一般都要先警告！让对方知道后果！

例外情况：来不及警告或警告会导致更严重后果的，可以直接使用！

🌰 举例：杀人犯正在砍人，每一秒都在威胁生命！你喊三遍住手可能人都死了！这时候来不及警告，直接开枪！

📌 四、使用前命令无关人员躲避（第6条）

使用警械武器之前，应当命令在场无关人员躲避！避免误伤！

🌰 举例：警察要开枪了，先喊：没关系的闪开！然后才能开枪。这就是为了避免误伤无辜！

记忆口诀：使用原则有两点：制止违法是目的，减少伤亡和财产！从轻到重层层递：口头→徒手→警械→武器！

⚠️ 易错点：警告不是必须的！当来不及警告或警告会导致更严重后果时，可以直接使用武器！

🛡️ 模块4: 警械使用情形

核心概念：警察在什么情况下可以使用警械？这是考试高频考点！

📌 一、驱逐性、制服性警械的使用（第7条）

前提条件：经警告无效！

可以使用驱逐性、制服性警械（警棍、催泪弹、高压水枪、特种防暴枪）的情况：

1. 结伙斗殴、殴打他人、寻衅滋事、侮辱妇女或进行其他流氓活动的

2. 聚众扰乱车站、码头、民用航空站、运动场等公共场所秩序的

3. 非法举行集会、游行、示威的

4. 强行冲越人民警察为履行职责设置的警戒线的

5. 以暴力方法抗拒或者阻碍人民警察依法履行职责的

6. 袭击人民警察的

7. 危害公共安全、社会秩序和公民人身安全的其他行为，需要当场制止的

🌰 举例：有人醉酒闹事打砸餐厅，警察来了喊他住手，他不听还打警察。这时候可以用警棍！因为：警告无效 + 袭击警察 + 以暴力抗拒执法！

📌 二、约束性警械的使用（第8条）

可以使用约束性警械（手铐、脚镣、警绳）的情况：

1. 遇有违法犯罪分子可能脱逃的

2. 遇有违法犯罪分子可能行凶的

3. 遇有违法犯罪分子可能自杀、自伤的

4. 遇有违法犯罪分子有其他危险行为的

5. 执行逮捕、拘留、看押、押解、审讯、拘传、强制传唤任务的

🌰 举例：抓到一个杀人犯，罪行很重，他可能要逃跑→可以戴手铐！抓到后要押送去看守所，他可能会自杀→可以戴手铐脚镣！

📌 三、使用限度（第7条第2款）

使用警械必须以制止违法犯罪行为为限度！一旦制止了，就立即停止使用！

🌰 举例：小偷已经被制服，手铐已经戴上了。这时候就不能再对他喷射催泪弹了！因为已经达到目的了！

记忆口诀：驱逐警械用7条，警告无效才用到！流氓斗殴闹游警，暴力抗法袭击警！约束警械戴手铐，脱逃行凶自杀跑！

⚠️ 易错点：使用警械的前提是警告无效！不是一上来就用！

💥 模块5: 武器使用情形（第9条）

核心概念：武器是致命性的！使用条件比警械严格得多！必须是严重暴力犯罪！

📌 一、使用武器的前提

1. 判明有暴力犯罪的紧急情形

2. 经警告无效

3. 来不及警告或警告会导致更严重后果的，可以直接使用

📌 二、可以使用武器的15种情形

1. 放火、决水、爆炸等严重危害公共安全的

2. 劫持航空器、船舰、火车、机动车或驾驶车船等机动交通工具，故意危害公共安全的

3. 抢夺、抢劫枪支弹药、爆炸、剧毒等危险物品，严重危害公共安全的

4. 使用枪支、爆炸、剧毒等危险物品实施犯罪或以使用这些物品相威胁实施犯罪的

5. 破坏军事、通讯、交通、能源、防险等重要设施，足以对公共安全造成严重、紧迫危险的

6. 实施凶杀、劫持人质等暴力行为，危及公民生命安全的

7. 国家规定的警卫、守卫、警戒的对象和目标受到暴力袭击、破坏或有受到袭击、破坏的紧迫危险的

8. 结伙抢劫或者持械抢劫公私财物的

9. 聚众械斗、暴乱等严重破坏社会治安秩序，用其他方法不能制止的

10. 以暴力方法抗拒或阻碍人民警察依法履行职责或暴力袭击人民警察，危及人民警察生命安全的

11. 在押人犯、罪犯聚众骚乱、暴乱、行凶或者脱逃的

12. 劫夺在押人犯、罪犯的

13. 实施放火、决水、爆炸、凶杀、抢劫或其他严重暴力犯罪行为后拒捕、逃跑的

14. 犯罪分子携带枪支、爆炸、剧毒等危险物品拒捕、逃跑的

15. 法律、行政法规规定可以使用武器的其他情形

🌰 举例：恐怖分子在火车站持刀砍人→可以用武器！因为：实施凶杀等暴力行为，危及公民生命安全！

📌 三、不用警告的情况

来不及警告，或者警告后可能导致更为严重危害后果的，可以直接使用武器！

🌰 举例：杀人犯正在拿刀砍人，你喊三遍住手他根本不听。这时候没时间警告了！直接开枪！

记忆口诀：武器使用条件严，暴力犯罪才能用！放火爆炸和劫持，凶杀抢劫杀警察！记住原则：能用警械就不用武器！

⚠️ 易错点：普通的小偷小摸、治安案件，不能用武器！只能用警械！

🚫 模块6: 不得使用武器的情形（第10条）

核心概念：武器是致命的！不到万不得已不能用！法律明确规定了两种不得使用武器的情形！

📌 一、不得使用武器的两种情形

情形一：犯罪分子是怀孕妇女或儿童

发现实施犯罪的人为怀孕妇女、儿童的，不得使用武器！

🌰 举例：一个怀孕的妇女在偷东西，警察抓到后不能对她开枪！但是要注意下面说的例外！

情形二：犯罪分子处于危险场所

犯罪分子处于群众聚集的场所或者存放大量易燃、易爆、剧毒、放射性等危险物品的场所的，不得使用武器！

🌰 举例：犯罪分子在加油站挟持人质。这时候不能用枪！因为一旦打中油罐，整个加油站都完了！

📌 二、例外情况（可以用武器！）

虽然有上面两个不得使用武器的情形，但有例外！

例外一：怀孕妇女、儿童使用枪支、爆炸、剧毒等危险物品实施暴力犯罪的，可以使用武器！

例外二：不使用武器予以制止，将发生更为严重危害后果的，可以使用武器！

🌰 举例：怀孕的妇女拿着炸弹要炸人群，这时候必须用武器！不炸，人群更惨！

📌 三、重要理解

为什么对怀孕妇女、儿童和危险场所特殊保护？

• 怀孕妇女和儿童属于弱势群体，社会应当给予特殊保护

• 危险场所使用武器可能造成二次伤害，后果更严重

• 但如果危害太大、不制止会更惨，就可以用了！

记忆口诀：不得用枪有两种：妇幼犯罪不能用，危险场所不能用！但是例外要记住：危险物品实施暴力，或者不用更惨的！

⚠️ 易错点：不是所有妇幼犯罪都不能用武器！如果她们使用枪支、爆炸、剧毒等危险物品，可以用武器！

⏹️ 模块7: 停止使用武器的情形（第11条）

核心概念：武器是用来制止犯罪的！一旦目的达到了，就要立即停止！不能继续打！

📌 一、应当立即停止使用武器的两种情形

情形一：犯罪分子停止实施犯罪，服从人民警察命令

当犯罪分子停止犯罪，并且服从警察的命令时，应当立即停止使用武器！

🌰 举例：小偷被警察拿枪指着，小偷说别开枪我不跑了，乖乖蹲下。这时候警察必须立即停止开枪！

情形二：犯罪分子失去继续实施犯罪的能力

当犯罪分子失去了实施犯罪的能力时，应当立即停止使用武器！

🌰 举例：杀人犯拿刀砍人，刀飞出去了，他手无寸铁了。这时候警察必须立即停止开枪！因为对方已经没有能力继续犯罪了！

📌 二、重要理解

为什么要规定停止使用武器的情形？

• 武器是用于制止犯罪的，不是用来惩罚的！

• 犯罪停止了，目的就达到了，不能再伤害对方

• 即使犯罪分子有罪，也应该由法院审判，不能由警察直接处决！

🌰 举例：张三拿刀砍人，警察开枪打中他的腿，张三倒地不起。这时候警察必须停枪！因为张三已经失去继续犯罪的能力了！

📌 三、停止使用后怎么办

停止使用武器后：

1. 应当立即停止射击

2. 应当及时抢救受伤人员（包括犯罪分子和无辜人员）

3. 应当保护现场

4. 应当立即报告

记忆口诀：停止用枪有两种：停止犯罪服从命，失去能力不能动！武器只是制止用，不是用来惩罚人！

⚠️ 易错点：不是犯罪分子放弃抵抗就可以停，是要停止犯罪 + 服从命令两个条件！

📝 模块8: 使用后的报告义务（第12条）

核心概念：使用武器后，必须立即报告！不能偷偷摸摸的！

📌 一、使用武器后的紧急处置

使用武器造成犯罪分子或者无辜人员伤亡的：

1. 及时抢救受伤人员（包括犯罪分子！）

2. 保护现场

3. 立即向当地公安机关或者该人民警察所属机关报告

🌰 举例：警察开枪打伤了一个持刀砍人的恐怖分子，打伤了旁边一个无辜路人。警察必须：1. 抢救恐怖分子 2. 抢救无辜路人 3. 保护现场 4. 立即报告！

📌 二、接到报告后的处置

当地公安机关或人民警察所属机关接到报告后：

1. 及时进行勘验、调查

2. 及时通知当地人民检察院（因为涉及犯罪！）

3. 及时通知伤亡人员家属或者所在单位

🌰 举例：区公安分局接到警察用枪的报告后，要派人去现场勘验，调查清楚怎么回事，然后通知检察院有人被枪打伤了，最后还要通知被打伤的人的家属！

📌 三、书面报告义务（第13条）

人民警察使用武器的，应当将使用武器的情况如实向所属机关书面报告！

注意：口头报告不行！必须是书面报告！

🌰 举例：警察用枪后，打个电话给领导说一声行不行？不行！必须写书面报告！写清楚：为什么用枪、怎么用的、结果怎么样！

📌 四、警械使用是否需要书面报告？

警械使用不需要书面报告！只有武器使用需要！

🌰 举例：警察用手铐铐了一个小偷，不用写报告。但如果警察开枪了，必须写书面报告！这就是武器和警械的区别！

记忆口诀：用枪之后四步走：抢救伤员保护场，报告领导和检察！书面报告不能少，警械不用写报告！

⚠️ 易错点：抢救的不只是无辜人员，犯罪分子也要抢救！

⚖️ 模块9: 法律责任（第14-15条）

核心概念：警察使用警械武器造成伤亡，涉及两个法律问题：赔偿和补偿！必须分清！

📌 一、违法使用警械武器（第14条）

人民警察违法使用警械武器，造成无辜人员伤亡或者财产损失的：

由该人民警察所属机关依照《国家赔偿法》的有关规定给予赔偿！

🌰 举例：警察在抓小偷时，小偷已经投降了，警察还把他打了一顿，打伤了。这叫违法使用！由警察局赔钱！

📌 二、依法使用警械武器（第15条）

人民警察依法使用警械武器，造成无辜人员伤亡或者财产损失的：

由该人民警察所属机关参照《国家赔偿法》的有关规定给予补偿！

🌰 举例：恐怖分子拿刀砍人，警察依法开枪打伤了他，同时误伤了一个路人。这是依法使用！由警察局补偿路人损失！

📌 三、赔偿和补偿的核心区别

对比项第14条（违法）第15条（依法）
使用性质违法使用依法使用
适用法律依照《国家赔偿法》参照《国家赔偿法》
结果类型赔偿补偿
性质有法律依据，必须赔没有法律依据，参照赔
金额全额赔偿酌情补偿

📌 四、依照vs参照（考试重点！）

依照：严格按照法律条文执行！有法可依！

参照：参考着执行，但不是必须按照！因为没有专门的法律规定怎么补偿！

🌰 举例：违法→依照→赔偿→有法必依！依法→参照→补偿→参考着办！考试经常考这两个词的区别！

记忆口诀：违法依照来赔偿，依法参照来补偿！依照必须有依据，参照只是参考！赔偿补偿要分清：违法=赔，依法=补！

⚠️ 易错点：依照和参照不能互换！违法用→依照→赔偿！依法用→参照→补偿！

🔫 警械武器分类

• 武器=枪支弹药（致命）

⚖️ 使用原则

• 目的：制止违法犯罪

• 要求：尽量减少伤亡

• 强度：从轻到重

• 顺序：口头→徒手→警械→武器

• 前提：一般要先警告

• 第7条：驱逐性警械（8种情形）

• 第8条：约束性警械（多种情形）

• 前提：警告无效

• 限度：以制止为限，立即停止

• 第9条：15种严重暴力犯罪
