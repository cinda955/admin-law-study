💰 国家赔偿法基础知识学习

系统学习国家赔偿法知识，轻松应对辅警考试

💵 模块1: 国家赔偿的概念

核心概念：国家赔偿就是国家机关及其工作人员违法行使职权，给公民造成损害后，由国家给予赔偿的制度！简单说就是：谁赔？国家赔！为什么赔？因为违法了！钱从哪来？纳税人的钱！

📌 一、什么是国家赔偿

想象一下：警察在执法过程中把你打伤了，谁来赔你？是打你的那个警察赔，还是警察局赔？答案是：警察局赔！国家赔！

• 谁来赔：国家（具体是侵权的机关，如公安局、法院）

• 为什么赔：因为国家机关或工作人员违法行使职权

• 钱从哪来：纳税人的钱！

🌰 举例：唐山相亲强奸案——某人被错误关押3年，最后获得57万国家赔偿。这57万从哪来的？都是纳税人的钱！为什么这样举例？因为考试经常考：国家赔偿的钱最终来源于财政收入，也就是我们纳税人交的钱！

📌 二、赔偿和补偿的区别（必背！）

违法 = 赔偿，合法 = 补偿！这是最核心的区别！

• 赔偿：是因为违法行为造成损失——赔！

• 补偿：是因为合法行为造成损失——补！

🌰 举例：警察违法拘留你→赔偿！政府合法征地修路占用你家土地→补偿！为什么这样举例？因为考试经常让区分赔偿和补偿，记住：违法=赔偿，合法=补偿！

📌 三、重要理解

虽然是一个警察打你，但实际上是国家在打你！因为警察是在行使国家权力！所以赔你的是国家，不是那个警察个人！

🌰 举例：警察打你，不是警察赔你钱，是警察背后的公安局赔！因为警察是代表国家在执法！为什么这样举例？因为很多人会误解为是警察个人赔，这是错的！

记忆口诀：国家赔偿三句话：违法行使职权造成损害，国家来赔钱！违法=赔偿，合法=补偿！国家赔的钱来自纳税人！

⚠️ 易错点：很多人以为国家赔偿是打人的警察赔，其实是国家赔！警察只是代表国家行使权力！

📋 模块2: 国家赔偿的类型

核心概念：国家赔偿分为两种类型！考试必考！

📌 一、行政赔偿

行政机关在行使行政职权时违法造成损失，由国家给予赔偿！

• 哪些机关？公安局、市场监管局、城管、生态环境局等行政机关

• 什么情况？违法行使行政职权造成老百姓损失

🌰 举例：城管在执法时违法扣押你的三轮车，造成损坏→行政赔偿！找城管局赔！为什么这样举例？因为行政赔偿是最常见的类型，考试经常考！

📌 二、司法赔偿

司法机关在行使司法职权时违法造成损失，由国家给予赔偿！

• 主要包括：刑事司法赔偿（最重要！）

• 哪些机关？法院、检察院、公安机关（侦查阶段）、看守所、监狱

🌰 举例：法院错误判决某人入狱3年，后发现是冤案→司法赔偿！找作出错误判决的法院赔！为什么这样举例？因为刑事司法赔偿是司法赔偿的重点！

📌 三、两种赔偿的对比

类型机关职权类型
行政赔偿行政机关（公安局、城管局等）行政职权
司法赔偿法院、检察院、看守所、监狱司法职权

记忆口诀：国家赔偿分两种：行政赔偿找行政机关，司法赔偿找司法机关！行政和司法，职权不同赔偿对象不同！

🔍 模块3: 构成要件

核心概念：国家赔偿要成立，必须同时满足4个要件！

📌 一、主体要件

必须是国家机关及其工作人员！普通个人打人不算国家赔偿！

• 国家机关：行政机关、司法机关

• 工作人员：在这些机关上班的人

• 受委托的组织：如联防队（受公安局委托），联防队侵权→委托的公安局赔！

🌰 举例：治安联防队打人了，谁赔？联防队赔不起，所以是委托他们的公安局赔！为什么这样举例？因为考试经常考受委托组织的赔偿问题！

📌 二、行为要件（关键！）

必须是职务行为！不是个人行为！

• 职务行为：在上班、执行公务时造成的损害→赔！

• 个人行为：下班后、跟工作无关的行为→不赔！

🌰 举例：民警在执法时发现情敌，一拳打过去→职务行为→赔！民警下班回家路上碰到情敌，打了一拳→个人行为→不赔！为什么这样举例？因为考试最爱考职务行为和个人行为的区分！记住：上班时间、执行公务时 = 职务行为 = 赔！下班后 = 个人行为 = 不赔！

📌 三、结果要件

必须造成人身权或财产权的损害！

• 人身权：生命、健康、人身自由、名誉等

• 财产权：财产、物品等

🌰 举例：被错误拘留3年→人身自由受损→赔！被违法罚款5000元→财产权受损→赔！为什么这样举例？因为没有损害结果就不赔！

📌 四、因果关系要件

违法行为和损害结果之间必须有因果关系！

🌰 举例：老太太摆摊，城管让她离开，她不走。城管说再不走就收东西，老太太一生气自己把水果推翻了。城管赔不赔？不赔！因为损害是老太太自己造成的，和城管没有因果关系！为什么这样举例？因为考试会考因果关系的判断！

记忆口诀：国家赔偿四要件：主体是国家机关，行为要看职务，结果要有损害，因果关系不能少！口诀：主体行为结果因果，缺一不可要牢记！

⚠️ 易错点：最容易错的是职务行为和个人行为的区分！上班时间、执行公务时是职务行为，下班后是个人行为！

✅ 模块4: 赔偿范围（赔 vs 不赔）

核心概念：这是考试超级重点！必须分清什么赔、什么不赔！

📌 一、赔的情形（侵犯人身权）

行政机关及其工作人员在行使职权时，有下列情形之一的，受害人有权获得赔偿：

• 违法拘留或违法采取限制人身自由的行政强制措施

• 非法拘禁或以其他方法非法剥夺公民人身自由

• 殴打、虐待或唆使、放纵他人殴打、虐待造成身体伤害或死亡

• 违法使用武器、警械造成身体伤害或死亡

🌰 举例：警察在审讯时殴打嫌疑人导致骨折→赔！警察依法使用武器击毙持刀行凶的恐怖分子→不赔（合法）！为什么这样举例？因为违法使用武器才赔，合法使用不赔！

📌 二、赔的情形（侵犯财产权）

• 违法实施罚款、吊销许可证和执照、责令停产停业、没收财物等行政处罚

• 违法对财产采取查封、扣押、冻结等行政强制措施

• 违法征收、征用财产

🌰 举例：市场监管局违法查封了你的店铺→赔！公安机关依法扣押了违法犯罪工具→不赔（合法）！为什么这样举例？因为违法查封扣押才赔，合法的不赔！

📌 三、不赔的情形（重要！）

不赔！千万记住！

• 个人行为：行政机关工作人员与行使职权无关的个人行为

• 受害人自己的行为：因公民、法人自己的行为致使损害发生

• 不可抗力/意外事件：如天降冰雹毁车（没尽保管义务→补偿，尽了→不赔不补）

• 自伤自残：因公民自伤、自残等故意行为致使损害发生

🌰 举例：民警在执法时收缴了小摊贩的三轮车，返回单位途中天降冰雹，三轮车被毁。民警尽到了妥善保管义务（躲闪不及）→不赔不补！如果民警没有把车停到指定位置就去吃饭，车被偷了→补偿！为什么这样举例？因为不可抗力+尽到义务=不赔不补；不可抗力+没尽到义务=补偿！

记忆口诀：不赔有四类：个人行为不赔，自己造成不赔，天灾尽了不赔不补，天灾没尽要补偿！口诀：个人自己天灾自残，四种情形国家不赔！

⚠️ 易错点：天灾（不可抗力）的情况最容易搞混！记住：尽到保管义务的天灾→不赔不补；没尽到义务的天灾→补偿！

🏢 模块5: 赔偿义务机关（考试必考！）

核心概念：找谁赔？找赔偿义务机关赔！这是考试超级重点！

📌 一、基本规则

行政机关及其工作人员行使职权侵犯公民权益造成损害的，该行政机关就是赔偿义务机关！

🌰 举例：公安局民警打伤了人→赔偿义务机关是公安局！城管局工作人员违法扣押了物品→赔偿义务机关是城管局！为什么这样举例？因为谁侵权谁就是赔偿义务机关！

📌 二、特殊情况

• 共同侵权：共同作为赔偿义务机关（可以向任何一个要求赔偿，那个机关应先予赔偿）

• 委托的组织：委托的行政机关（如联防队→公安局）

• 复议案件：最初造成侵权的机关（复议加重的话，复议机关对加重部分赔）

• 机关撤销：继续行使职权的机关；没有的话，撤销它的机关

🌰 举例：市场监管局和公安局共同没收假烟→两个局都是赔偿义务机关！联防队打人→找公安局赔！复议案件复议机关加重了处罚→最初侵权的机关赔，复议机关赔加重部分！为什么这样举例？因为考试经常考各种特殊情况！

📌 三、派出机构（特别重要！）

派出所不能当赔偿义务机关！设立它的公安局才是！

🌰 举例：派出所警察打人了，向谁申请赔偿？派出所！错！应该是派出所背后的公安局！为什么这样举例？因为派出所虽然能当被告，但不能当赔偿义务机关！这是特别容易搞混的点！

📌 四、司法赔偿义务机关

• 拘留：作出拘留决定的机关

• 逮捕：作出逮捕决定的机关（检察院）

• 再审改判无罪：作出原生效判决的法院

• 二审改判无罪：作出一审有罪判决的法院

🌰 举例：检察院批准逮捕某人，后判无罪→检察院赔！法院一审判决有罪，二审改判无罪→一审法院赔！为什么这样举例？因为司法赔偿的义务机关和行政赔偿不一样！

记忆口诀：赔偿义务机关找谁赔？记住口诀：单独侵权单独赔，共同侵权共同赔，委托侵权找委托，派出机构找设立，复议案件找原机关，机关撤销找接管！

⚠️ 易错点：派出所不能当赔偿义务机关！这是考试超级爱考的点！派出机构侵权→找设立它的机关赔！

📝 模块6: 赔偿程序

核心概念：申请国家赔偿有两种方式和时间要求！

📌 一、赔偿请求的提出

• 单独提出：先向赔偿义务机关申请（先行处理），2年内提出

• 一并提出：在申请行政复议或提起行政诉讼时一并提出

🌰 举例：你被公安局违法拘留了，你可以单独向公安局申请赔偿（2年内）；也可以在告公安局的行政复议或诉讼中一并要求赔偿！为什么这样举例？因为两种方式都可以！

📌 二、时间要求（要背！）

程序时间说明
申请时效2年内从知道侵权之日起计算，被羁押期间不计算
处理期限2个月内赔偿义务机关作出是否赔偿的决定
送达决定10日内作出赔偿决定后10日内送达
起诉期限3个月内对赔偿决定不服，向法院起诉

📌 三、协商

赔偿义务机关可以与赔偿请求人就赔偿方式、赔偿项目、赔偿数额进行协商！但是是否赔偿不能协商！

🌰 举例：赔偿义务机关说赔你5万，你想要8万，可以协商！但不能说我不赔你钱你给我道歉——是否赔偿不能协商！为什么这样举例？因为考试会考协商的范围！

记忆口诀：赔偿程序时间多，2年申请2月决，10日送达3月诉！协商只管方式项目数额，是否赔偿不能协商！

⚖️ 模块7: 司法赔偿（刑事司法赔偿）

核心概念：刑事司法赔偿超级重要！要搞清楚拘留、逮捕、判决什么情况赔！

📌 一、拘留的赔偿（重点！）

刑事拘留分三种情况：

• 违法拘留→赔！（违反刑事诉讼法规定）

• 合法拘留+超期羁押+最后无罪→赔超期部分！

• 合法拘留+没超期+最后无罪→不赔！（公安机关依法拘留，查清事实后认为不需追究刑事责任）

🌰 举例：警察依法拘留了张三，拘留期限届满前查清事实，发现抓错人了→不赔！因为警察是依法拘留，没有违法也没有超期。但如果拘留超期了才放人→赔超期部分！为什么这样举例？因为合法拘留不超期不赔，只有超期才赔！这是容易搞混的点！

📌 二、逮捕的赔偿

逮捕后决定撤销案件、不起诉、判决无罪→赔！

简单说：逮捕后无罪→赔！逮捕后有罪→不赔！

🌰 举例：检察院逮捕了李四，后判无罪释放→赔！检察院逮捕了王五，后判有罪→不赔！为什么这样举例？因为逮捕后无罪说明错了，错了就要赔！

📌 三、再审改判无罪

依照审判监督程序再审改判无罪，原判刑罚已执行→赔！

• 再审改判无罪：原生效判决的法院赔

• 二审改判无罪：一审法院赔

🌰 举例：张三被判10年，执行了5年后再审改判无罪→赔！李四一审被判5年，二审改判无罪→一审法院赔！为什么这样举例？因为谁做的错误判决谁赔！

📌 四、不赔的情形

• 因公民自己故意作虚伪供述被羁押

• 不负刑事责任的人被羁押（不满14周岁、完全不能辨认行为能力）

• 依照刑事诉讼法规定不追究刑事责任的人被羁押

• 工作人员个人行为

• 自伤自残

记忆口诀：刑事赔偿记清楚：违法拘留要赔，合法拘留超期赔，没超期不赔；逮捕无罪赔，逮捕有罪不赔！口诀：拘留看超期，逮捕看有罪，判决看改判！

💰 模块8: 赔偿方式和标准

核心概念：怎么赔？赔多少？这些数字要背！

📌 一、赔偿方式

• 以支付赔偿金为主（最主要的赔偿方式）

• 能返还财产或恢复原状的，予以返还或恢复

• 赔偿金不纳税！（重要考点！）

🌰 举例：国家赔你57万，这57万不用交个人所得税！为什么这样举例？因为考试会考国家赔偿金是否纳税——不纳税！

📌 二、人身自由赔偿标准

侵犯公民人身自由的，每日赔偿金 = 国家上年度职工日平均工资

计算方式：年平均工资 ÷ 法定工作日

📌 三、生命健康权赔偿标准（要背！）

伤害程度赔偿项目标准
身体伤害医疗费+护理费+误工费误工费按日平均工资计算，最高5倍
部分丧失劳动能力医疗费+护理费+残疾赔偿金等最高不超过年平均工资10倍
全部丧失劳动能力同上+生活费（被扶养人）年平均工资20倍
死亡死亡赔偿金+丧葬费总额=年平均工资20倍

🌰 举例：警察殴打公民致死→赔死亡赔偿金+丧葬费，总额为国家上年度职工年平均工资的20倍！为什么这样举例？因为死亡赔偿是固定金额（20倍），不是按实际损失算！

📌 四、精神损害赔偿

致人精神损害的：消除影响、恢复名誉、赔礼道歉

造成严重后果的：还要支付精神抚慰金

严重后果包括：羁押6个月以上、轻伤以上、残疾、精神障碍等

🌰 举例：唐山相亲强奸案当事人被错误关押3年，名誉受损→除了赔57万，还要在报纸上登报道歉！为什么这样举例？因为精神损害赔偿要区分一般损害和严重后果！

记忆口诀：赔偿标准要记牢：身体伤害最高5倍，部分丧失10倍，死亡全部20倍！精神损害要区分，一般道歉，严重加钱！赔偿金不用纳税！

🔙 模块9: 追偿

核心概念：国家先赔，然后向有故意或重大过失的工作人员追偿！

📌 一、追偿的条件

• 赔偿义务机关已经支付了赔偿金

• 工作人员有故意或重大过失

• 先赔后追：国家先赔受害人，再向责任人追偿

🌰 举例：民警在执法时故意殴打嫌疑人致伤，国家赔了10万给受害人。国家可以向该民警追偿！为什么这样举例？因为民警故意打人，必须追偿！

📌 二、追偿的范围

• 追偿金额不超过国家赔偿的数额

• 不包括处理赔偿请求时支出的费用（交通费、误工费等）

🌰 举例：国家赔了10万，追偿时最多只能要回10万，不能多要！处理赔偿请求的差旅费不能追偿！为什么这样举例？因为追偿不能超过实际赔偿金额！

📌 三、追偿后的处理

• 责令有故意或重大过失的工作人员承担部分或全部赔偿费用

• 构成犯罪的，还要追究刑事责任

• 应当给予处分

🌰 举例：警察刑讯逼供打死人，国家赔偿后向该警察追偿。警察除了赔钱，还要被开除、判刑！为什么这样举例？因为追责是全方位的：赔钱+处分+判刑！

📌 四、司法追偿范围

• 殴打虐待犯罪嫌疑人造成伤亡

• 违法使用武器警械造成伤亡

• 徇私舞弊、枉法裁判

记忆口诀：追偿就是国家先赔后要回，故意重大过失才能追，追偿金额不超赔偿额，追责还有处分判刑！

📋 知识框架总览

💵 国家赔偿概念

国家机关违法行使职权给公民造成损害

核心：违法=赔偿，合法=补偿

钱来自纳税人

📋 两种赔偿类型

行政赔偿：行政机关侵权

司法赔偿：司法机关侵权

主要是刑事司法赔偿

🔍 四大构成要件

主体要件：国家机关及工作人员

行为要件：职务行为（不是个人）

结果要件：人身权/财产权损害

因果关系：行为与损害有关

✅ 赔偿范围

赔：违法拘留、殴打、违法使用武器

赔：违法罚款、查封、扣押、征用

不赔：个人行为、自己造成、天灾+尽义务

🏢 赔偿义务机关

谁侵权谁就是义务机关

共同侵权共同赔

委托找委托方

派出所不能当义务机关！

📝 赔偿程序

申请时效：2年内

处理期限：2个月内

起诉期限：3个月内

协商：方式/项目/数额

⚖️ 司法赔偿

拘留：违法赔/超期赔/合法不超不赔

逮捕：无罪赔/有罪不赔

再审改判：原审法院赔

💰 赔偿标准

方式：支付赔偿金为主

身体伤害：最高5倍

部分丧失：最高10倍

死亡/全部丧失：20倍

精神损害：道歉+抚慰金

赔偿金不纳税！

🔙 追偿

先赔后追

故意/重大过失才追

不超过赔偿额

追责：赔钱+处分+判刑
