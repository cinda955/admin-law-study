⚖️ 行政诉讼法基础知识学习

民告官到法院，零基础小白也能轻松上手！

📜 模块1: 行政诉讼概念

核心概念：行政诉讼就是"民告官"，但是要到法院去告！

📌 一、什么是行政诉讼？

行政诉讼 = 民告官到法院

• 民（公民、法人）告官（行政机关）

• 告到人民法院（不是复议机关）

• 法院审查行政行为的合法性

🌰 举例：张三被公安局拘留10天，张三不服，可以到法院起诉公安局！为什么这样举例？因为行政诉讼就是民告官到法院，和行政复议的区别就是：一个告到机关，一个告到法院！

📌 二、行政诉讼的特点

• 被告恒定：只能是行政机关，老百姓永远是原告

• 审查单一：只审查合法性，不审查合理性（这是和复议的重要区别！）

• 两审终审：可以上诉，不像复议那样一锤定音

• 不得调解：审理行政案件不适用调解（但行政赔偿、行政补偿和自由裁量权案件除外）

🌰 举例：张三被罚款2000元，张三到法院起诉公安。法院怎么判？只审查罚款合不合法，不审查罚款2000元重不重（合理性）！为什么这样举例？因为考试常考"诉讼只查合法性，复议既查合法性又查合理性"！

记忆口诀：行政诉讼：民告官到法院，只查合法不查合理，两审终审不得调解！口诀：去法院告！法院只管合不合法，不管处罚重不重！

⚠️ 易错点：行政复议既查合法性又查合理性，行政诉讼只查合法性！这是两者的核心区别！

📋 模块2: 基本原则

核心概念：行政诉讼的基本原则要记牢！

📌 一、独立审判原则

人民法院依法独立行使审判权

• 法院独立，不受行政机关、社会团体和个人干涉

• 检察院、法院、监察委员会都是独立行使职权

🌰 举例：张三告公安局，法院怎么判？法院自己说了算，县长、市长都不能干涉！为什么这样举例？因为宪法规定法院独立审判，不受行政机关干涉！

📌 二、合法性审查原则（重点！）

行政诉讼只审查合法性，不审查合理性！

• 合法 = 有没有法律依据、程序对不对

• 合理 = 处罚轻重是否适当（诉讼不管这个）

🌰 举例：张三违章停车被罚款200元，张三嫌贵告到法院。法院会不会说"罚款200太贵了，改成50"？不会！法院只审查罚款有没有法律依据、程序对不对，不会管200元是重还是轻！为什么这样举例？因为考试必考"诉讼只查合法性"！

📌 三、当事人地位平等原则

原被告法律地位平等！

• 进入法院后，老百姓和行政机关平等

• 行政复议不一样，复议机关是上级，和被申请人不太平等

🌰 举例：张三告公安局，在法院开庭时，张三和公安局局长平等对话，公安局不能以"我是官"压人！为什么这样举例？因为复议是官管官，法院是民告官要平等！

📌 四、辩论原则

当事人可以辩论！

• 原告、被告可以当庭辩论

• 复议没有辩论程序（书面审查为主）

🌰 举例：法院开庭时，张三说"我没违法"，公安局说"你有视频证据"，双方当庭辩论！复议没有这个环节！为什么这样举例？因为诉讼和复议的审理方式不同！

记忆口诀：基本原则：独立审判、合法审查（只查合法）、地位平等、辩论原则！口诀：法院说了算，只管合不合法！老百姓和官在法院面前平等！

⚠️ 易错点：诉讼只查合法性，不查合理性！复议既查合法又查合理！

🎯 模块3: 复议与诉讼的区别

核心概念：复议和诉讼都是民告官，但有大区别！

📌 一、受理机关不同（最本质区别！）

• 行政复议：受理机关是行政机关（上级机关）

• 行政诉讼：受理机关是人民法院（法院）

🌰 举例：张三被县公安局罚款，张三不服。找县政府申请复议（行政复议），还是找法院起诉（行政诉讼）？都可以！找县政府是复议，找法院是诉讼！为什么这样举例？因为这是两条不同的告官途径！

📌 二、审查强度不同

• 行政复议：既审合法性又审合理性

• 行政诉讼：只审合法性

🌰 举例：张三被罚款2000元，复议机关可以说"罚款太重了，改成500"（审合理性）！但法院只能说"罚款有依据，合法"或者"罚款违法，撤销"（只审合法性）！为什么这样举例？因为复议是官管官可以改，诉讼是法院只管合法不合法！

📌 三、程序衔接（重要！）

• 复议前置：有些案件必须先复议，不能直接诉讼

• 复议后诉讼：复议不服可以诉讼

• 诉讼后不复议：诉讼是终局的，不能再复议！

🌰 举例：自然资源确权案件，必须先申请行政复议，不服才能到法院诉讼！为什么这样举例？因为涉及专业性强的领域要先复议！

📌 四、复议前置的情形

• 自然资源确权

• 纳税争议

• 工伤保险待遇

• 治安拘留（警告、罚款）

🌰 举例：张三被行政拘留15天，他能不能直接去法院告？不能！必须先申请行政复议，不服才能诉讼。为什么这样举例？因为考试常考"复议前置"！

记忆口诀：复议与诉讼：复议到机关告，诉讼到法院告；复议双重审查，诉讼单一审查；诉讼终局不能再复议！口诀：找政府复议，找法院打官司，完全两码事！

⚠️ 易错点：诉讼是终局的，诉讼之后不能再复议！因为法院独立行使审判权，不受行政机关干涉！

📄 模块4: 受案范围

核心概念：哪些行政行为可以告？考试必考！

📌 一、可以诉讼的范围（和复议基本相同）

以下行政行为可以提起行政诉讼：

• 行政处罚：罚款、拘留、警告等

• 行政强制：查封、扣押、冻结等

• 行政许可：颁发许可证、执照等

• 行政确认：确认土地、房屋权属等

• 行政征收：征收税款、土地等

• 行政给付：发放最低生活保障费等

🌰 举例：张三被公安局罚款5000元，可以到法院起诉！张三的房屋被城管强制拆除，可以到法院起诉！为什么这样举例？因为这些都属于可诉的行政行为！

📌 二、不能诉讼的范围（重要！）

以下行为不能提起行政诉讼：

• 国防外交行为

• 抽象行政行为（行政法规、规章、规范性文件）

• 内部行政行为（奖惩、任免公务员）

• 行政机关最终裁决

• 刑事司法行为（取保候审、逮捕等）

• 行政调解、行政指导

• 驳回重复申请的行政行为

🌰 举例：张三告国务院？不能告！国务院的决定是最终裁决，不能告！乡政府发布的"征地补偿标准"？不能单独告！这是抽象行政行为，只能附带审查！为什么这样举例？因为考试常考"不能告的情形"！

📌 三、和复议范围的细微差别

诉讼范围比复议范围稍窄：

• 工伤认定：可以复议，但不能诉讼

• 政府信息公开：可以复议，但不予公开的信息不能诉讼

🌰 举例：张三申请工伤认定，被人社局驳回，张三可以申请行政复议，但不能到法院诉讼！为什么这样举例？因为考试会区分复议和诉讼的范围差异！

记忆口诀：可诉范围：处罚强制许可确认征收给付！不可诉：国防抽象内部刑事调解指导！口诀：罚款拘留能告，但红头文件和内部任命不能单独告！

⚠️ 易错点：抽象行政行为不能单独诉讼，但可以附带审查！就是说起诉具体行政行为时，可以要求法院审查附带的相关规范性文件！

🏛️ 模块5: 级别管辖

核心概念：告哪个法院？基层还是中级？考试必考！

📌 一、级别管辖概述

• 基层人民法院：管辖大多数案件（最忙！）

• 中级人民法院：管辖特殊案件（重点记忆！）

• 高级人民法院：管辖省内重大复杂案件

• 最高人民法院：管辖全国重大复杂案件

🌰 举例：告派出所、告县公安局、告市政府——哪个到基层告？派出所和县公安局到基层告！市政府到中级告！为什么这样举例？因为考试常考"告谁去哪个法院"！

📌 二、中级人民法院管辖范围（必须背！）

记住三种情形：

1. 对国务院部门或者县级以上人民政府作出的行政行为提起的诉讼

2. 海关处理的案件（不管什么级别）

3. 本辖区内重大复杂的案件、重大影响案件、共同诉讼案件、涉港澳台案件

🌰 举例：告公安部？中级法院！因为公安部是国务院部门！告广东省公安厅？基层法院！因为省公安厅不是国务院部门！告深圳市政府？中级法院！因为市政府是县级以上政府！为什么这样举例？因为考试常考"中级法院管辖范围"！

📌 三、简单记忆法

• 乡政府 → 基层法院

• 县政府 → 基层法院

• 市政府 → 中级法院

• 省政府 → 中级法院

• 公安部 → 中级法院

• 省公安厅 → 基层法院

• 海关（不分级别） → 中级法院

🌰 举例：记住口诀："除乡政府外，所有政府都中级"！部门和海关特殊记：国务院部门+海关=中级，省级部门=基层！为什么这样举例？方便记忆！

记忆口诀：级别管辖：基层管大多数，中级管特殊！口诀：告部委和县级以上政府，去中级法院！其他全部去基层！

⚠️ 易错点：告省政府要到中级法院！告省公安厅反而到基层法院！因为省公安厅是省级部门，不是国务院部门！看行政级别，不是看"省"字！

🌍 模块6: 地域管辖

核心概念：在哪个地方的法院告？

📌 一、一般地域管辖（原告就被告）

由最初作出行政行为的行政机关所在地法院管辖

• 原告就被告：到被告所在地法院告

• 方便法院调查取证

🌰 举例：张三被深圳市公安局罚款，张三应该到深圳市的法院起诉，而不是跑到北京去告！为什么这样举例？因为被告在哪里，就到哪里告！

📌 二、经复议案件的地域管辖（特殊！）

可以由原机关所在地或复议机关所在地法院管辖

• 经复议的案件：可以选择

• 可以到原机关所在地法院告

• 也可以到复议机关所在地法院告

🌰 举例：张三被县公安局罚款200元，张三申请行政复议到市政府，市政府维持了。张三可以去县法院告，也可以去市法院告，选择一个！为什么这样举例？因为复议案件可以选择管辖法院！

📌 三、限制人身自由的特殊规定（辅警必须掌握！）

可以选择被告所在地或原告所在地法院告！

• 被告所在地法院

• 原告户籍所在地法院

• 原告经常居住地法院

• 被限制人身自由地法院

🌰 举例：张三在广州被北京警方拘留，张三可以到北京的法院告，也可以到自己老家的法院告（户籍地、经常居住地、限制人身自由地）！为什么这样举例？因为辅警工作中涉及限制人身自由，这个知识点很重要！

📌 四、不动产案件专属管辖

因不动产提起的诉讼，由不动产所在地法院管辖

• 查封、扣押不动产 → 不动产所在地法院

• 拆除违章建筑 → 不动产所在地法院

🌰 举例：张三的房子在深圳，被城管强制拆除，张三应该到深圳的法院起诉（不动产所在地）！为什么这样举例？因为不动产案件由不动产所在地法院专属管辖！

记忆口诀：地域管辖：原告就被告！复议案件可选择！限制人身自由四选一！不动产专属管！口诀：一般去被告那告，复议后可以换地方告！被拘留了随便选！

⚠️ 易错点：复议案件地域管辖：不管复议维持还是改变，都可以选择原机关所在地或复议机关所在地！但只能选一个，不能两个都选！

👥 模块7: 诉讼参加人

核心概念：谁可以告（原告）？谁可以被告（被告）？

📌 一、原告（谁可以告？）

行政行为的相对人或者利害关系人

• 行政相对人：直接被处罚、被管理的张三

• 利害关系人：邻居、竞争对手等间接受影响的人

• 官（行政机关）不能当原告

🌰 举例：张三建房子，住建局发许可证给张三，邻居李四觉得影响采光。邻居可以告吗？可以！因为李四是利害关系人。为什么这样举例？因为考试常考"利害关系人可以当原告"！

📌 二、被告（告谁？）

作出行政行为的机关或者组织

• 行政机关：公安局、市场监管局等

• 法律授权的组织：高校、公立医院等

• 被委托的组织：委托的机关是被告

🌰 举例：治安联防队抓人，被告是公安局，不是联防队！因为联防队是公安局委托的，被委托的不是被告。为什么这样举例？因为考试常考"被委托的不是被告"！

📌 三、第三人（谁有利害关系？）

同案件有利害关系的人

• 可以申请参加诉讼

• 法院可以通知其参加诉讼

• 第三人有独立的诉讼地位

🌰 举例：张三告公安局拘留他，李四是被张三打的受害人。李四可以作为第三人参加诉讼。为什么这样举例？因为第三人是有利害关系的人！

📌 四、特殊情形

• 合伙组织：以登记的字号为原告，推选代表

• 个体工商户：有字号以字号为原告，无字号以经营者为原告

• 股份制企业：股东大会等可以企业名义起诉

🌰 举例：美宜家便利店被罚款，原告是"美宜家"（字号），不是老板张三！如果没有字号，原告才是张三。为什么这样举例？因为个体工商户的原告确认有讲究！

记忆口诀：诉讼参加人：原告=民（相对人或利害关系人），被告=官（作出行为的机关或组织），第三人=有利害关系的人！口诀：记住，民告官！谁告谁是原告，谁被告谁是官！

⚠️ 易错点：被委托的组织不是被告，委托的机关才是被告！联防队、城管协管员等都是被委托的，不是被告！

⚖️ 模块8: 被告确定规则

核心概念：告谁？复议后被告怎么确定？考试超级重点！

📌 一、直接诉讼的被告

谁作出行政行为，谁就是被告

• 派出所罚款 → 告派出所

• 公安局拘留 → 告公安局

• 县政府决定 → 告县政府

🌰 举例：派出所对张三罚款200元，张三直接到法院起诉，被告是派出所！为什么这样举例？因为直接诉讼，谁作行为告谁！

📌 二、复议后诉讼的被告（重点！三种情况！）

记住三种情形：

1. 复议维持：共同告（原告+复议机关一起告）

2. 复议改变：单独告（只告复议机关）

3. 复议不作为：选择告（告原机关或复议机关都行）

🌰 举例：张三被县公安局罚款200元，申请行政复议到市政府：
• 市政府维持原处罚 → 共同告：张三告县公安局+市政府
• 市政府改变罚款为500元 → 单独告：张三只告市政府
• 市政府逾期不作决定 → 选择告：张三可以选择告县公安局或市政府
为什么这样举例？因为这是考试超级重点！

📌 三、"改变"的判断

• 改变处理结果：如罚款从200改为500

• 撤销原行为

• 确认原行为无效

• 注意：只改变依据但结果没变 = 维持！

🌰 举例：原行为罚款200元，复议改变为罚款100元 → 改变！原行为罚款200元，复议依据从第10条改为第15条但还是罚200元 → 维持！为什么这样举例？因为判断是否"改变"要看结果！

📌 四、共同被告的特殊情形

• 共同作出行政行为的机关是共同被告

• 告漏了：法院通知原告追加，原告不同意追加的 → 列为共同被告

🌰 举例：张三被公安局和市场监管局联合处罚，张三只告公安局。法院通知追加，市场监管局作为共同被告。为什么这样举例？因为共同行为要共同负责！

记忆口诀：被告确定：直接诉讼谁作谁告；复议后：维持共同告，改变单独告，不作为选择告！口诀：复议维持找两个机关告，改变只告复议机关，不作为随便挑一个！

⚠️ 易错点：复议维持=共同被告（原告+复议机关都要告），复议改变=只告复议机关！不能搞混！

⏰ 模块9: 起诉期限与举证

核心概念：多久内可以告？谁来举证？

📌 一、起诉期限（多久内可以告？）

一般情况：知道行政行为之日起6个月内

• 直接起诉：6个月

• 经过复议后起诉：15日

• 不知道行政行为：最长1年（知道之日起6个月，但不超过知道后1年）

🌰 举例：张三1月1日被公安局罚款，张三申请行政复议，3月1日复议决定作出。张三应该在3月16日前到法院起诉（复议决定送达后15日内）！为什么这样举例？因为复议后再起诉期限只有15天！

📌 二、被告举证（谁来举证？）

被告（行政机关）承担举证责任！

• 被告要证明行政行为合法

• 原告一般不需要举证

• 但在以下情况原告要举证：

- 证明自己申请过

- 证明自己受到损害

🌰 举例：张三告公安局拘留他违法，张三要不要证明拘留违法？不需要！是公安局要证明拘留合法！为什么这样举例？因为官举证是原则！民告官，还是官举证！

📌 三、证据保全

• 诉前证据保全：当事人可以在起诉前申请

• 诉讼中证据保全：法院可以依申请或依职权保全

• 保全证据不得少于2人

🌰 举例：张三的店铺被城管查封，店里有监控录像。张三怕证据灭失，可以申请证据保全！为什么这样举例？因为保全证据很重要！

记忆口诀：起诉期限：直接6个月，复议后15天！举证：被告举证，原告特殊举证！口诀：直接告6个月，复议后再告只有15天！证明你有理是官的事！

⚠️ 易错点：经过复议的案件，起诉期限是15日，不是6个月！复议是15天，诉讼是6个月！

📝 模块10: 审理与判决

核心概念：多久审完？怎么判？

📌 一、审理程序

• 普通程序：6个月内审结（可延长）

• 简易程序：45日内审结（不能延长）

• 简易程序适用：事实清楚、权利义务明确、争议不大的案件

🌰 举例：张三对罚款50元不服起诉，法院适用简易程序，45日内要判决！为什么这样举例？因为简易程序快！

📌 二、第二审程序（上诉）

• 上诉期限：判决15日内，裁定10日内

• 二审审限：3个月内作出终审判决

• 两审终审：二审判决是终审判决

🌰 举例：一审判决后，张三上诉，二审法院3个月内要作出终审判决！为什么这样举例？因为二审有期限限制！

📌 三、判决类型

根据不同情况作出不同判决：

1. 维持判决：行政行为正确，合法

2. 撤销判决：行政行为违法

3. 履行判决：行政机关不履行法定职责

4. 变更判决：行政处罚明显不当（只审合理性例外）

5. 确认违法判决：行为违法但不宜撤销

🌰 举例：法院判决撤销公安的拘留决定 → 撤销判决！公安不履行法定职责 → 履行判决！为什么这样举例？因为判决类型要区分！

📌 四、行政首长出庭应诉

告官要见官！

• 行政机关负责人应当出庭应诉

• 不能出庭的应当说明情况

• 可以委托1-2名诉讼代理人

🌰 举例：张三告公安局，公安局局长应当出庭！如果实在来不了，要说明情况。为什么这样举例？因为告官要见官！

📌 五、不适用调解（特殊！）

• 审理行政案件不适用调解

• 例外：行政赔偿、行政补偿、自由裁量权案件可以调解

🌰 举例：张三告公安局罚款2000元，法院不能调解！但如果是行政赔偿案件，双方可以调解。为什么这样举例？因为调解有例外！

记忆口诀：审理期限：普通6月，简易45天，二审3月！判决类型：维持撤销履行变更确认！口诀：一审普通半年，简易45天，二审三个月！

⚠️ 易错点：二审期限是3个月，不是6个月！一审普通程序才是6个月！

📋 知识框架总览

1. 行政诉讼概念

核心：民告官到法院，只查合法不查合理

口诀：告官到法院，只查合法！

易错：诉讼只查合法，复议双重审查！

2. 基本原则

核心：独立审判、合法审查、地位平等、辩论

口诀：独立审判只查合法，地位平等能辩论！

易错：只查合法性，不查合理性！

3. 复议与诉讼区别

核心：复议到机关告，诉讼到法院告

口诀：复议告机关，诉讼告法院！

易错：诉讼终局不能再复议！

4. 受案范围

核心：处罚强制许可确认征收给付可诉

口诀：处罚强可诉，抽象内部不可诉！

易错：抽象行为可附带审查！

5. 级别管辖

核心：基层管大多数，中级管特殊

口诀：国务院部门县级以上，海关全部中级管！

易错：省公安厅到基层，省政府到中级！

6. 地域管辖

核心：原告就被告，复议可选

口诀：一般原告就被告，复议可选，限制自由随便选！

易错：不动产专属管辖！

7. 诉讼参加人

核心：原告是民，被告是官

口诀：原告是民，被告是官！

易错：被委托的不是被告！

8. 被告确定规则

核心：维持共同告，改变单独告，不作为选择告

口诀：维持共同，改变单独，不作为随便！

易错：复议维持=共同被告！

9. 起诉期限与举证

核心：直接6个月，复议后15天，被告举证

口诀：直接六个月，复议半月，被告举证！

易错：复议后起诉只有15天！

10. 审理与判决

核心：普通6月，简易45天，二审3月

口诀：普通六个月，简易四十五，二审三个月！

易错：二审是3个月！
